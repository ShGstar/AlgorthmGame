# AlgorthmGame
服务器端游戏中常见算法

1.跳跃表

#### 概念
是基于有序链表的扩展，提取出链表中关键节点作为索引的多层链表，以空间换时间提高查找性能。
#### 时间及空间复杂度

| 算法 | 平均 | 最差 |
| ------ | ------ | ------ |
| 空间 | O(n) | O(n log n) |
| 搜索 | O(log n) | O(n) |
| 插入 | O(log n) | O(n) |
| 删除 | O(log n) | O(n) |

#### 应用
1.Redis的sorted set

2.LevelDB的memtable

3.排行榜
